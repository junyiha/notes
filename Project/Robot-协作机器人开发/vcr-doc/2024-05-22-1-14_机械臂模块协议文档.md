+ 新版本下的机械臂模块，基于TICP的通讯协议

## 请求数据的基本结构

```
| version | command | device id | json data |
```

+ version : 通讯协议版本号
+ command : 命令码
+ device id : 设备ID
+ json data : 具体命令请求的参数

## 应答数据的基本结构

```json
{
    "version":1,
    "error_number":0,
    "message":"success"
}
```

+ version : 应答通讯协议版本号
+ error_number : 通讯错误码
+ message : 帮助信息
+ ... : 根据不同的请求，返回的不同的应答数据
