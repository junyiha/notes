## 框架设计

+ 设计思路：
  + 使用 sqlite3 持久化存储数据
  + 使用 mongoose 与前端交互
  + 使用 nlohmann/json 处理JSON数据

+ 需要存储的数据
  + 摄像头：名称，流地址
  + 算法数据：算法名，文件路径
  + 任务：名称，配置信息
  + 事件：任务名称，摄像头名称，算法名称，标记状态，事件时间
  + 盒子信息：盒子名称，设备型号，注册状态，序列号，软件版本，设备状态，系统时间，网络设置
